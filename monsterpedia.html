01. OVERVIEW // 游戏概念企划

在开发初期，我们通过 One Pager 快速统一了团队的愿景：一款强调线索推理与心理博弈的 2D 点击解谜游戏。

02. THE PITCH // 超自然调查与社交博弈

《Monsterpedia》是一款融合了超自然调查与社交战斗（Social Combat）的游戏。玩家扮演一名侦探，必须揭开看似普通犯罪背后的真实谜团。

  • 现场勘查与线索收集： 调查犯罪现场，收集证据，并与旁观者交谈以还原真相。
  • 社交破防机制 (Composure System)： 游戏的核心战斗不是物理攻击，而是通过玩家收集的证据来反驳嫌疑人的主张，从而逐步削弱他们的“心理防线 (Composure)”，最终打破伪装抓住真凶。

03. ARCHITECTURE & POSTMORTEM // 架构重构与开发复盘

作为该项目的开发者，我深刻理解了游戏系统必须为扩展性和团队协作做好规划。在开发过程中，我主导了底层架构的重构，解决了以下核心问题：

  • 突破耦合瓶颈： 在添加新游戏机制时，原有的整体代码架构面临巨大挑战。我设计了基于 Scriptable Object 的数据与逻辑分离架构，彻底降低了线索与对话系统的耦合度。
  • 工业化效率提升： 花时间重组系统极大地改善了整体工作流，减少了长期问题。重构后，新增一个线索分支的配置时间从 30 分钟压缩至 20 分钟（效率提升 30%），显著提升了团队迭代效率。

"通过这个项目，我最大的收获是对整体开发流程的把控感。—— William Su"
