DUALING DRIVERS

2D 像素双人坦克对抗与关卡编辑器

2024.01 - 2024.05 5人团队 玩法策划 & 程序实现
Controller InputLevel EditorJSON 读写C# / Unity

01. GAME FEEL // 拟真与街机：双轨驾驶模型

为了同时满足休闲玩家与硬核玩家的需求，我设计并用 C# 实现了两套截然不同的手柄底层输入逻辑：

  • 街机模式 (Arcade)： 采用绝对方向映射，左摇杆直接控制坦克移动方向，极大地降低了新手的上手门槛。
  • 拟真履带模式 (Simulation)： 深度模拟真实坦克驾驶体验。左摇杆控制左侧履带，右摇杆控制右侧履带。极具机械感的操作逻辑为核心玩家提供了巨大的操作上限与沉浸感。

02. LEVEL EDITOR // JSON 读写与 UGC 开放

开放玩家 UGC（用户生成内容）是我们拓展游戏寿命的核心策略。我为此开发了内置的网格地图编辑器。

  • 自定义地图搭建： 支持放置陆地、水面（减速/不可通行）、可破坏墙壁（提供战术掩体）与不可破坏墙壁。
  • 数据序列化： 实现了地图数据的本地存取功能。玩家绘制的地图可以被序列化保存为独立文件，并在主菜单中随时被重新读取和加载。

03. PROGRESSION // 关卡体系与内容广度

除了功能强大的编辑器，游戏本身也内置了数个设计精良的官方关卡。选关界面不仅管理着官方关卡，也充当着读取玩家自定义地图文件的集线器（Hub）。

  • 官方关卡： 每个关卡都有独特的地形博弈与道具（加速、修复、弹夹增幅）生成点布局。
  • 无限重玩： 编辑器与选关系统的打通，让 Dualing Drivers 拥有了无限的游戏可能性。
