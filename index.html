FILE // 档案

苏省

System & Level Designer

我是苏省，罗彻斯特理工学院（RIT）游戏设计专业学生，预计2027年毕业，目前主要方向是系统策划与关卡设计。

我主要使用 Unity 进行开发，熟悉 ScriptableObject 数据结构、基础系统设计以及关卡节奏设计。在项目中通常负责玩法系统设计、关卡流程规划和部分工具配置。

我比较喜欢策略和模拟类游戏，例如《十字军之王》《环世界》《Kenshi》等，这些游戏对系统结构和玩家决策空间的设计对我影响很大。目前也在尝试设计自己的策略与SLG玩法系统。

曾在网易雷火参与关卡策划实习，参与过战斗关卡流程搭建和关卡事件逻辑设计。希望未来能继续从事系统或关卡策划方向的工作。

PHILOSOPHY // 开发者画像

■ 心流与节奏控制 (Pacing & Flow)： 7 年的京剧与古琴舞台经验，让我对观众的情绪起伏极其敏感。我擅长将这种张力控制转化为关卡中的玩家情感曲线（Emotion Curve），在设计中精准把控张弛度。

■ 高密度空间设计 (Spatial Design)： 关卡设计如同在方寸间做道场。在《Sailors》等项目中，我专注于在有限的物理空间内，通过严密的尺度控制和路线规划，为玩家创造高信息密度的策略纵深。

■ 系统驱动与数据严谨 (System-Driven Design)： 作为重度大战略游戏（如 P社系列）玩家，我对“系统涌现性”与“秩序感”有强烈追求。无论是构建底层逻辑，还是像烘焙时死磕 1°C 温差一样去推敲数值模型，我始终致力于打造稳定可控的设计系统。
